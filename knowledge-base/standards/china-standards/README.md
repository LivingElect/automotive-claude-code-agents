# China Automotive Standards Knowledge Base

## Overview

This directory contains reference materials for China's intelligent connected vehicle (ICV) standard system, including mandatory national standards (GB), recommended standards (GB/T), and regulatory guidance documents.

## Directory Structure

```
china-standards/
├── README.md                          # This file
├── gb-l2-adas-compliance.md          # L2 combined driving assistance mandatory standard
├── gb-l3-ads-requirements.md         # L3 automated driving system requirements
├── gb-parking-standards.md           # Parking assistance and automated parking
├── gbt-functional-safety.md          # GB/T 34590 (ISO 26262 adoption)
├── gbt-cybersecurity.md              # GB/T 40857 (ISO 21434 adoption)
├── china-sotif-application.md        # SOTIF application in Chinese context
├── miit-access-management.md         # MIIT product access process
├── china-test-scenarios.md           # China-specific test scenario library
└── regional-regulations.md           # Pilot city and provincial regulations
```
