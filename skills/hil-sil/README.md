# HIL/SIL/VIL Skills

Comprehensive Hardware-in-the-Loop (HIL), Software-in-the-Loop (SIL), and Vehicle-in-the-Loop (VIL) testing skills for automotive embedded systems.

## Skill Categories

### Core Testing (10 skills)
- hil-setup.yaml - HIL testbench configuration
- sil-test.yaml - Software-in-the-Loop testing
- vil-test.yaml - Vehicle-in-the-Loop testing
- test-automation.yaml - Automated test execution
- can-bus-testing.yaml - CAN bus testing
- fault-injection.yaml - Fault injection testing
- sensor-simulation.yaml - Sensor simulation
- carla-integration.yaml - CARLA simulator integration
- gazebo-integration.yaml - Gazebo/ROS 2 integration
