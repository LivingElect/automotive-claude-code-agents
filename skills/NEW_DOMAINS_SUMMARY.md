## File Structure & Quality

Each skill file contains:
- **Metadata**: name, version, category, domain, subcategory, description
- **Use Cases**: 5 practical automotive applications
- **Automotive Standards**: 4 relevant industry standards
- **Instructions**: 80-150 lines of comprehensive content including:
  - Core competencies (material science, regulatory landscape, strategic frameworks)
  - Technical deep-dives (properties, methodologies, workflows)
  - Approach (10-step implementation process)
  - Deliverables (reports, documentation, test results)
  - Best practices (industry-proven guidelines)
  - Integration with automotive systems
- **Constraints**: 4 real-world limitations
- **Tools Required**: 5 industry-standard tools/software
- **Tags**: Category-specific keywords for searchability
