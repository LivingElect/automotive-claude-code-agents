# Migration Guide: Vector CANape → ETAS INCA

## Overview

Migrating from **CANape** to **INCA**

- **Source**: ECU calibration and measurement tool supporting XCP, CCP protocols
- **Target**: Calibration, diagnostics, and validation tool for ECUs

## Feature Comparison

| Feature | Source | Target |
|---------|--------|--------|
| API Available | ✅ | ✅ |
| Scripting | Python | Python |
| Price Tier | enterprise | enterprise |

## Migration Steps

1. **Assessment Phase**
   - Inventory all CANape projects and configurations
   - Identify custom scripts and automations
   - Document current workflows

2. **Preparation Phase**
   - Install INCA
   - Set up licensing and user accounts
   - Create test environment

3. **Migration Phase**
   - Convert project files
   - Migrate configurations
   - Port automation scripts

4. **Validation Phase**
   - Test all migrated projects
   - Verify automation workflows
   - Train team on new tool

## Common Challenges

- Configuration format differences
- Script language migration
- Workflow adaptation
